Answer 1

Show me how to calculate the average price of all items.

items.reduce((total, current) => total + current.price/items.length, 0);

Answer 2

Show me how to get an array of items that cost between $14.00 and $18.00 USD

items.filter((item) => 14 < item.price && item.price < 18).map((item) => `${item.title}`);

Answer 3

Which item has a "GBP" currency code? Display it's name and price.

Answer 4

Display a list of all items who are made of wood.

items.filter((item) => item.materials.includes("wood")).map((item) => `${item.title}`);

Answer 5

Which items are made of eight or more materials? Display the name, number of items and the items it is made of.

items.filter((item) => item.materials.length >= 8).map((item) => `${item.title} has ${item.materials.length} materials:
${item.materials.join('
')}`);
